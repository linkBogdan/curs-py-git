# curs-py-git

Un curs gratuit și voluntar dedicat elevilor de liceu care vor să învețe bazele programării în **Python** și colaborarea prin **Git & GitHub**.

## Obiectiv
Să îi ajutăm pe elevi să înțeleagă principiile de bază ale programării și să învețe cum se lucrează în echipă la proiecte software reale, folosind instrumentele pe care le utilizează și profesioniștii.

## Conținut
Acest repository conține:
- Exemple practice folosite în timpul cursului  
- Scripturi Python simple pentru exerciții  
- Ghiduri pas cu pas pentru Git și GitHub  
- Materiale pentru proiectele finale

## Structura cursului
1. Introducere în Git și GitHub  
2. Colaborare și documentare (Markdown, Pull Requests)  
3. Bazele Python — variabile, condiții, bucle  
4. Funcții și structuri de date  
5. Mini-proiecte și introducere în AI  

## Pentru cine este
Pentru elevii de liceu fără experiență anterioară în programare, dar curioși să descopere lumea dezvoltării software.

## Despre proiect
Inițiativă voluntară, educațională și non-profit, destinată să promoveze învățarea practică și colaborativă.
