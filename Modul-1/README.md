# Modul 1: Git & GitHub - Primii pasi ca developer

Un modul dedicat introducerii in lumea programatorilor.

---

## Cuprins:
In acest modul instalam Git pe Windows, invatam comenzi de baza precum:
- **clone, pull, push, commit**

Flag-uri precum:
- **-m sau -am**

Cat si comenzi de navigare prin bash:
- **cd, mv, touch**

Pe langa asta discutam despre diferenta intre developarea pe Windows si cea pe Linux cat si alte subiecte introductive.
